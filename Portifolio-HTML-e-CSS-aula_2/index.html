Este projeto é um portfólio pessoal criado com HTML, CSS e JavaScript, representando os primeiros passos no universo do desenvolvimento web.

Ele busca ser uma vitrine dinâmica das minhas habilidades e do meu progresso como desenvolvedor front-end. O design simples e direto permite que visitantes conheçam mais sobre mim, minha paixão pela tecnologia e os primeiros projetos que venho desenvolvendo.

A página apresenta links para minhas redes sociais profissionais, como GitHub e Instagram, permitindo um contato mais direto com aqueles que quiserem acompanhar minha jornada.
